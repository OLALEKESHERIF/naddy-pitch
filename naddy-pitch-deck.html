// MILESTONES_&_ROADMAP

Three tranches.
Three deliverables.

TRANCHE 1 — DAY 1

Payment Link MVP

Deploy the core payment link engine. Merchants generate unique USDT/USDC deposit addresses tied to a specific payment ID and amount. Hosted payment page shows deposit address, token, and expected amount. Live on Ethereum Sepolia.

TRANCHE 2 — DAY 2

Merchant REST API

Full FastAPI layer goes live. Merchants integrate via a single POST /payments/create endpoint. Returns payment link and deposit address in milliseconds. Includes full Postman collection and API documentation.

TRANCHE 3 — DAY 3

Webhook & Confirmations

On-chain payment monitor and HMAC-SHA256 signed webhook dispatcher. Polls blockchain every 30 seconds, detects confirmed transfers, marks payments confirmed, fires webhook with full payment data to merchant callback URL.
